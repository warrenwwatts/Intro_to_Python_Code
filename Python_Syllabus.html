Python an Introduction - Course Syllabus


1. Course Description

Python an Introduction is an introduction to computer programming using the Python language. Absolute beginners or students already familiar with other programming languages will learn how to write Python programs that cover a wide range of applications.


This course will meet every Monday from 6pm to 9pm from Nov 5 to Jan 14. The class will be a blend of readings, lectures, coding demonstration, and hands on projects. Expect to spend 3-5 hours each week on assigned reading and exercises outside class.

3. Text and References

The text for the course is a free E-book Python for Everybody – by Charles R. Severance https://www.py4e.com/lessons


4. Required Software

We will use (free) open source software. In this class we will use Spyder IDE from Anaconda. Please download Anaconda Python 3.6 or 3.7 version before the class at https://www.anaconda.com/download/


5. Required Classwork

Homework required to pass the class will be CodeAcademy’s “Learn Python” free course. Please sign up for an account. https://www.codecademy.com/learn/learn-python
